# История разработки проекта

## 25 марта 2026 - Начало проекта

### Идея проекта
Создать мобильное приложение для детей 3-14 лет с виртуальными персонажами.
Ребёнок может позвонить персонажу по видео-звонку, персонаж слышит ребёнка и отвечает голосом.

### Ключевые решения
1. **Бюджет**: 0 рублей, делаем MVP для привлечения инвестиций
2. **Команда**: Я (разработчик/маркетолог) + AI Assistant
3. **Персонаж MVP**: Только Баба Яга
4. **Технологии**:
   - Silero STT/TTS (бесплатно, open-source)
   - GigaChat API (бесплатные токены для теста)
   - Expo (упрощённая разработка)
   - React Native + TypeScript
   - Python + FastAPI

### Архитектурные решения
- Мультиязычность с самого начала (i18n)
- Образовательный компонент - в v2.0
- Вместо сложной анимации - минимальная анимация картинки
- Ребёнок видит себя на экране (как в мессенджере)

### Важные ссылки и ресурсы
