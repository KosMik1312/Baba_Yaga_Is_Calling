# Baba Yaga Is Calling - Детское AI приложение 📱

Мобильное приложение для видео-звонков детей со сказочными персонажами.

## 🎯 Описание

MVP версия приложения с одним персонажем - Бабой Ягой.

**Основные возможности:**
- ✅ Ребёнок видит себя на экране (как в мессенджере)
- ✅ Анимированная Баба Яга отвечает голосом
- ✅ Распознавание речи ребёнка
- ✅ AI-диалог через GigaChat
- ✅ Голосовой ответ через Silero TTS

## 🛠 Технологический стек

### Frontend
- **React Native** + **TypeScript**
- **Expo** (с возможностью eject для WebRTC)
- **Zustand** - состояние
- **React Navigation** - навигация
- **i18next** - интернационализация

### Backend
- **Python** + **FastAPI**
- **WebSocket** - реальное время
- **SQLite** (для MVP) → PostgreSQL для продакшена

### AI Сервисы
- **GigaChat API** - диалоги (бесплатные токены для теста)
- **Silero STT** - распознавание речи (бесплатно, open-source)
- **Silero TTS** - синтез голоса (бесплатно, open-source)

## 🚀 Быстрый старт

### Мобильное приложение
```bash
cd mobile
npm install
npx expo start
```

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
