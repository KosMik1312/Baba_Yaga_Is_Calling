# 📋 Roadmap

## ✅ Этап 1: MVP (завершён)

- [x] Структура проекта
- [x] React Native + Expo SDK 54
- [x] FastAPI + WebSocket сервер
- [x] GigaChat API интеграция
- [x] Whisper STT — распознавание русской речи
- [x] Silero TTS — синтез голоса Бабы Яги
- [x] Полный AI пайплайн: аудио → текст → GigaChat → голос
- [x] WebSocket соединение фронт ↔ бэкенд
- [x] Toggle-запись микрофона (нажал — запись, нажал — отправил)
- [x] Воспроизведение ответа на фронте
- [x] Поддержка веб-браузера (конвертация WebM → WAV через ffmpeg)
- [x] История диалога (последние 10 сообщений в контексте)
- [x] uv для управления Python окружением
